Diabetes Prediction Using Logistic Regression

Project Overview :-

This project focuses on predicting diabetes risk using Machine Learning techniques and healthcare data. 
The objective was to analyze patient health indicators, identify key factors associated with diabetes, and build a predictive model capable of classifying individuals based on their diabetes status.

Diabetes Dataset :-

The dataset contains 100,000 patient records with features such as:

Gender
Age
Hypertension
Heart Disease
Smoking History
BMI
HbA1c Level
Blood Glucose Level
Diabetes (Target Variable)

Project Workflow :-

Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
Data Visualization using Matplotlib and Seaborn
Categorical Feature Encoding
Correlation Analysis using Heatmaps
Feature Scaling
Logistic Regression Model Development
Model Evaluation using Accuracy Score, Confusion Matrix, and Classification Report
Feature Importance Analysis

Key Insights :-

The analysis revealed that HbA1c Level, Blood Glucose Level, BMI, and Age are among the most influential factors associated with diabetes prediction.

Technologies Used :-

Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Conclusion

This project provided hands-on experience in the complete Machine Learning pipeline, from data preprocessing and exploratory analysis to model development and performance evaluation.
It demonstrates the practical application of Logistic Regression for healthcare analytics and predictive modeling.
